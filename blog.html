How I Started My Web Development Journey

Web development has helped me understand how the Internet works, how websites are structured, and how I can turn ideas into real web applications.I built my very first multi-page website using HTML5. I learned how to structure pages using semantic elements like <header>, <nav>, <main>, and <footer>, and how to link them all together with anchor tags.

The most exciting moment was clicking between pages for the first time and watching the navigation actually work. It felt like assembling a puzzle and seeing the full picture emerge. Web development and data analysis have more in common than I expected — both are about building structure from scattered parts.

Learning CSS Styling

CSS transformed my plain HTML pages into something I am truly proud of. Today I worked with colours, fonts, margins, and the box model. I kept thinking about how similar this is to formatting a Power BI report — presentation matters just as much as the data behind it.

I applied a consistent colour scheme, added Google Fonts, and used Flexbox to centre elements cleanly. The most satisfying part was watching my navigation bar go from a stack of plain links to a sleek, styled menu.
